# Archive Directory

This directory contains historical files from the project development process.

## Structure

- **training_scripts/** - Old training script versions (v1-v7)
- **logs/** - Historical training logs
- **docs/** - Old documentation and analysis files
- **pid_files/** - Old process ID files
- **models/** - Pretrained model files

## Current Active Files

The current working files are in the project root:
- `train_v7_final_working.py` - Current stable training script
- `training.pid` - Active training process ID
- `training_WORKERS0_*.log` - Current training log (in logs/)

## Note

These files are kept for reference and debugging purposes but are not part of the active training workflow.
